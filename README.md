ShareDemo
=========

微信分享到朋友、朋友圈；QQ分享

ShareDemo是为iOS的APP提供社会化功能的一个组件

一、1.0版主要功能：

    1、支持分享到微信好友、微信朋友圈、QQ好友

    IOS版本以上，IOS 5.0 以上。
    

二、URL Scheme说明
１）QQ URL Scheme，QQ + 十六进制新AppId。（如 appid=222222 则 scheme=QQ0003640E）
２）WeiChat URL Scheme  为appId，（如 appid=wxd930ea5d5a258f4f 则 scheme=wxd930ea5d5a258f4f）
